  • Бывает так, что нужно показать не только скринлист, а ещё например обложку этого видео, или эскизы скриншотов из видео (а их несколько) или, учитывая, что хостинги дохнут хотелось бы и альтернативные скринлисты видеть, но SpoilerListGenerator позволяет вставлять только одну-единственную ссылку для одного видео заменяя текст %IMG%, что не очень удобно.

    Данная фича позволяет исправить этот недостаток.

    Просто отредактируйте свой шаблон, там где хотите, чтобы были другие ссылки пишите %IMG2% %IMG3% (%IMG1% это то, что будет вместо %IMG%) и т.д.

    Примечание: если вы хотите показать несколько эскизов (или картинок) в ряд, писать нужно только один раз.

  • К сожалению, spoiler list generator в текущей версии не может сгенерировать результат без ссылок, поэтому снимаем галочку "Принимать только ссылки", и вместо ссылок копируем текст из поля и жмем кнопочку "Вставить" в SpoilerListGenerator (там 1000 элементов, если надо больше - просто жмем "Вставить" ещё раз)

  • Важно!
    %IMG1% должен использоваться только один раз на один элемент (видео), для правильного определения количества элементов и сопоставления с количеством ссылок в дальнейшем
